# DISHES - HEXOCEAN TASK

Made for HexOcean by Wojciech Glid  

## VERSIONS

- Node 16.16.0  
- NPM 8.19.2

## PROJECT SETUP

1. Go to the project directory and run `npm install`
2. Run `npm run dev` to start dev server on port *5173*, or `npm run build` to build for production
